# Essential Git Commands

1. `git init` – Initialize repository  
2. `git clone` – Clone a repository  
3. `git status` – Check file status  
4. `git add` – Stage changes  
5. `git commit` – Save changes  
6. `git push` – Upload changes  
7. `git pull` – Fetch updates  
8. `git branch` – Manage branches  
9. `git checkout` – Switch branches  
10. `git merge` – Merge branches
